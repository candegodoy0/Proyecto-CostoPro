# Anexo - “Frameworks y Node JS”

Este anexo complementa la documentación del proyecto "CostoPro" con una investigación sobre dos herramientas ampliamente utilizadas en el ecosistema de desarrollo web moderno: **React** y **ExpressJS**. Ambos frameworks fueron seleccionados por su popularidad, su sólida documentación, y su aplicabilidad a proyectos similares al desarrollado.

El objetivo es analizar sus características, beneficios potenciales para el proyecto, nivel de dificultad y utilidad práctica.

- [Anexo - ExpressJS](./Anexo-ExpressJS.md)
